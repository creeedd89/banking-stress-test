# Banking Stress Test Analysis

This repository contains tools for fetching and analyzing banking data to perform stress testing, risk analysis, and event studies.

## Files
- `fetch_data.py`: Script to fetch the necessary data.
- `event_study.py`: Tool for performing event studies on the data.
- `risk_analysis.py`: Analyzes the risk metrics.
- `generate_report.py`: Generates the final analysis report.
- `requirements.txt`: Python dependencies required to run the scripts.

## Setup
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run scripts as needed.
